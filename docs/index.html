Qubic Tic Tac Toe

Inspired by one of my highschool coding projects.


How to Play

This is a standard game of tic tac toe, but you have to get four in a row, and it's in 3D. Connecting four in a row can be done horizontally on or through planes, vertically on or through planes, or diagonally on or through planes, or some combination. Visually it can be difficult to see, but writing out the coordinates can be helpful.

Players take turns placing either an X or an O within the cube attempting to get four in a row. The Player interface consists of four 'boards,' each representing a 'slice' of the cube as displayed below.

Games can be won by connecting four in a row horizontally across a board, vertically, or diagonally. Connecting four in a row across multiple slices is allowed too.

If you notice any bugs or issues with the game, please do not hesitate reach out to me or open an issue on GitHub. Also, single player currently has only one difficulty, and it's hard. Good luck :)
